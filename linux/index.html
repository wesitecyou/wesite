Google Chrome

  1. 下载并安装 Google 的签名密钥

    2. wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo gpg --dearmor -o /usr/share/keyrings/google-chrome-keyring.gpg

  2. 添加 Google Chrome 的软件源

    3. echo "deb [signed-by=/usr/share/keyrings/google-chrome-keyring.gpg] http://dl.google.com/linux/chrome/deb/ stable main" | sudo tee /etc/apt/sources.list.d/google-chrome.list

  3. 更新软件包列表

    4. sudo apt update

  4. 安装 Google Chrome

    5. sudo apt install google-chrome-stable
